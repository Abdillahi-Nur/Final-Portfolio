TETRIS GAME


HTML5
CSS3
JavaScript

This is my JavaScript take on the classic game 'Tetris'.

This project demonstrates a range of JavaScript skills, including manipulating the DOM, the Math Object, functions and if statements, predominantly using arrays.

The game is operated by using the four keycodes and the player scores 10 points for each row cleared. The user is not able to move right when the shape is placed far right, and vice versa for a shape place on the left.

The functionality is currently set on the tetromino moving down a square every 1 seconds.
