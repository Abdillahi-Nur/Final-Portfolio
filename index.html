GREENLANE MASJID HOMEPAGE CLONE

HTML, CSS, JavaScript

GREENLANE MASJID HOMEPAGE CLONE


HTML5
CSS3
JavaScript

This is my clone of the Apple homepage (UK version early September 2020).

It primarily consists of HTML5 and vanilla CSS (using flexbox and the grid system).

The hamburger menu and the footer links were the main sources of JavaScript - both on the mobile and tablet views. The table in the footer collapses into an accordion on small screens, JavaScript was used to mirror the original.

Every screen break included an image change for all of the cards. No frontend frameworks were used during this project and this ended up being a pure "look and recreate" task - i.e. for 99% of this job I refrained from looking at the dev tools on the original webite to make this a difficult challenge.

On completion I was very happy with this outcome.
